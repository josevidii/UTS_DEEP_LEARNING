# README

## Cara Menjalankan Notebook

1. Download atau clone repository GitHub ini ke komputer Anda.

2. Pastikan Python dan Jupyter Notebook / Google Colab sudah tersedia.

3. Install library yang dibutuhkan secara manual menggunakan:

```bash id="5k0mda"
pip install pandas numpy matplotlib seaborn scikit-learn tensorflow
```

4. Buka file notebook (`.ipynb`) menggunakan:

* Jupyter Notebook
* JupyterLab
* VS Code
* atau Google Colab

5. Jalankan seluruh cell secara berurutan dari atas ke bawah.

## Catatan

* Pastikan dataset berada pada path/folder yang sesuai dengan notebook.
* Notebook deep learning disarankan dijalankan menggunakan GPU agar training lebih cepat.
* Seluruh preprocessing, training, dan evaluasi model akan berjalan otomatis setelah notebook dijalankan.

* Kasus 1 : Data Tabular
* Kasus 2 : Digit Recognizer
* Kasus 3 : NLP
